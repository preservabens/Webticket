Guia de Padrões e Arquitetura do Front-End

Este documento serve como um guia para o desenvolvimento de novas páginas e funcionalidades, garantindo consistência e manutenibilidade.

1. Estrutura Principal (Single Page Application - SPA)
   - O arquivo `dashboard.html` é o contêiner principal da aplicação.
   - O arquivo `main.js` centraliza toda a lógica de navegação, manipulação de estado e eventos.
   - O conteúdo das páginas é carregado dinamicamente dentro do elemento `<main id="content-target">`.

2. Menu de Navegação Lateral (`#sidebar`)
   - O menu possui três estados principais, controlados por classes no `<body>`:
     a) Desktop - Expandido: Estado padrão em telas grandes. Largura de `250px`.
     b) Desktop - Recolhido: Ativado pela classe `sidebar-collapsed`. Mostra apenas ícones, com largura de `80px`.
     c) Mobile - Oculto: Estado padrão em telas menores (`<= 768px`). O menu fica fora da tela (`transform: translateX(-100%)`).
     d) Mobile - Aberto: Ativado pela classe `sidebar-open`. O menu desliza para dentro da tela.

3. Botão de Toggle do Menu (☰)
   - Nome da classe: `.page-menu-toggle-btn`.
   - Localização: Deve estar presente no cabeçalho de CADA página carregada (`tarefas.html`, `ponto.html`, etc.), dentro do `.page-title-group`, à esquerda do título `<h2>`.
   - Visibilidade:
     - Em telas grandes (> 768px): O botão SÓ aparece se o menu lateral estiver recolhido (`.sidebar-collapsed`).
     - Em telas pequenas (<= 768px): O botão aparece O TEMPO TODO.
   - Ação: Ao ser clicado, aciona a função `handleMenuToggle()` em `main.js`, que decide se deve recolher (desktop) ou abrir/fechar (mobile) o menu.

4. Comportamento do Menu ao Navegar
   - Ao clicar em um item do menu de navegação (`.nav-btn`):
     - Em telas de celular, o menu deve se fechar completamente.
     - Em telas de computador, se o menu estiver expandido, ele deve se recolher (encurtar).

5. Padrão para Novas Páginas
   - Toda nova página (ex: `nova_pagina.html`) deve seguir a estrutura de cabeçalho padrão para garantir a presença do botão de menu.
   - Exemplo de estrutura mínima para o cabeçalho da página:
     ```html
     <div class="page-header">
       <div class="page-title-group">
         <button type="button" class="page-header-btn page-menu-toggle-btn">☰</button>
         <h2>Título da Página</h2>
       </div>
       <!-- Outros botões de ação da página aqui -->
     </div>
     ```
